API Docs

This is the overview of the API. Provide a general description of what your API does and any important information that users need to know to get started.

GET /product

Returns a list of products.

GET /api/product

Parameters:

  • none

Return:

[
    {
        "id": 1,
        "name": "americano",
        "price": 50000
    }
]

POST /product

Create new product

POST /api/product

Body:

  • name string (required) - product name
  • price integer (required) - product price
{
    "name": "macha",
    "price": 100000
}

Return:

{
    "productid": 7
}

PATCH /product/:productid

Update a product

PATCH /api/product/:productid

Parameter:

  • productid (required) - Product ID

Body:

  • name string (optional) - new product name
  • price integer (optional) - new product price
{
    "name": "matcha",
    "price": 100000
}

Return:

{
    "result": "Success"
}

DELETE /product

Delete a product.

DELETE /api/product/:productid

Parameters:

  • orderid (required) - Product ID

Return:

{
    "message": "success"
}

POST /order

Create a new order

POST /api/order

Body:

  • name string (required) - customer name
  • phone string (required) - customer phone number
  • orders array (required) - array of order object:
    • id integer - product id (see GET /product)
    • quantity integer - product order quantity
{
    "name": "john",
    "phone": "0123123123",
    "orders": [
        {
            "id": 1,
            "quantity": 4
        },
        {
            "id": 3,
            "quantity": 5
        }
    ]
}

Return:

{
    "orderid": 14
}

PATCH /order/:orderid

Update an order

PATCH /api/order/:orderid

Parameter:

  • orderid (required) - Order ID

Body:

  • status integer (optional) - new order status (see GET /status)
  • orders array (optional) - array of order object:
    • id integer - product id (see GET /product)
    • quantity integer - new product order quantity
{
    "status": 3,
    "orders": [
        {
            "id": 3,
            "quantity": 0
        }
    ]
}

Return:

{
    "result": "Success"
}

DELETE /order/:orderid

Delete an order

DELETE /api/order/:orderid

Parameter:

  • orderid (required) - Order ID

Return:

{
    "result": "Success"
